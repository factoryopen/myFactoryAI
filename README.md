# myFactoryAI

#### 介绍

myFactoryAI：Agent驱动的工厂AI化运作平台 | 开源智能制造系统

myFactoryAI是一套完整的工厂业务系统（PLM/ERP/MES/APS/SRM/CRM/IIoT/车间数字孪生），以自主Agent体系为智能引擎，驱动传统业务一体化运作。

内置AI应答系统，支持自然语言对话、智能图表、数据洞察。通过四层Agent Skill体系，构建自需求、自开发、自建构、自运维的AI闭环
